Linux(1)--基本概念

发表于 2017-02-10 | 分类于 Linux

shell简介

编程语言分类

  1. 运行方式
  • 编译运行：源代码 -> 编译 -> 程序文件
  • 解释运行：源代码 -> 边解释边运行
  1. 编程功能
  • shell编程：利用系统上的命令和编程组件进行编程
  • 完整编程：利用库和编程组件进行编程
  1. 编程模型
  • 面向过程：以指令为中心，数据服务代码
  • 面向对象：以数据为中心，围绕数据组织

shell：解释运行，依赖于外部程序文件的过程式编程
shell脚本实际上是命令的堆积，但由于很多命令不具有幂等性，需要逻辑来判断运行条件是否满足，以避免命令执行发生的错误

Hexo命令

发表于 2017-02-03 | 分类于 github

简写

hexo n = hexo new
hexo p = hexo publish
hexo g = hexo generate
hexo s = hexo server
hexo d = hexo deploy
hexo c = hexo clean

新建

hexo new blog BLOG
hexo new page PAGE

服务器

hexo server
监视文件变动，自动更新

hexo server -s
静态模式

hexo server -p PORT
以指定端口启动

hexo server -i IP
以指定IP启动

hexo d -g
生成静态页面并部署
