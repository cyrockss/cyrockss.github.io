Linux(1)--基本概念

发表于 2018-03-10 | 分类于 Linux

本文介绍Linux的基本概念

程序：数据 + 指令

Linux内核：98% + 2%汇编

Linux程序版本：
Alpha：内测版
Beta：公测版

Linux基本法测

  1. 一切皆文件
  2. 体积短小且目的单一的程序
  3. 组合小程序完成复杂任务
  4. 尽可能避免捕获用户接口
  5. 配置数据保存至文本文件

操作系统作用

  • CPU时间分配
  • 内存管理
  • 进程管理
  • 安全功能
  • 驱动硬件

Linux文件名称

  1. 严格区分大小写
  2. 除了换行符均可使用，但不建议使用 空白 / ? *
  3. 不依赖扩展名区分文件
  4. 目录也是文件，目录不能跟文件同名

基本术语：
GUI：Graphic USer Interface
GLI：Command Line Interface
API：Application Program Interface
ABI：Application Binary Interface

常见GUI：
Gnome：C开发
KDE：C++开发
Xface：轻量级

终端：
物理终端：/dev/console
虚拟终端：/dev/tty(1,6)
模拟终端：/dev/pts/#
图形终端

手机内核：
Android：Linux + Java虚拟机
iOS：FreeBSD + ObjectC

命令：
内部命令：shell自带
外部命令：在某个路径下的可执行程序

工作路径：
默认工作路径为当前路径
