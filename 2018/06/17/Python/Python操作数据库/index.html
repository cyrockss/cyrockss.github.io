常见错误

Unable to acquire Oracle environment handle
解决办法：
将oracle客户端(例如：instantclient_12_1)加到环境变量PATH中

DatabaseError: DPI-1050: Oracle Client library is at version 0.0 but must be at version 11.2 or higher
解决办法：
下载cx_Oracle.exe安装包安装，注意python的版本号和位数
下载地址：https://pypi.python.org/pypi/cx_Oracle/5.3

DatabaseError: DPI-1047: 64-bit Oracle Client library cannot be loaded:
解决办法：
重新下载64位的oracle客户端(instantclient)
下载地址：
https://www.oracle.com/technetwork/database/database-technologies/instant-client/downloads/index.html
